# CODSOFT-task3 # Movie Genre Description System 🎬

## Overview

This project implements a simple **Movie Genre Description System** using Python.
The program allows users to enter the name of a movie and returns the **genre description** associated with that movie.

The system demonstrates a basic concept of **content-based information retrieval**, where each movie is mapped to its specific genre information.

## Features

* Accepts user input for movie names
* Displays the genre description for the selected movie
* Includes popular **Kannada movies**
* Simple and easy-to-use command-line interface
* Fast lookup using a Python dictionary

## Technologies Used

* Python

## Supported Movies

The system currently supports the following movies:

* KGF
* Kantara
* Rajakumara
* Martin
* KD
* Tagaru

## Example Output

User Input: kgf

Output:
Genre: action gangster gold crime

User Input: kantara

Output:
Genre: action thriller culture forest

User Input: rajakumara

Output:
Genre: family drama inspiration

## How to Run the Project

1. Install Python on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program using the following command:

python movie_genre_system.py

5. Enter the movie name when prompted.
